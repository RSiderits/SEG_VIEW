KILN SEGMENT VIEWER RAMP-HOLD CHART

PURPOSE
This program allows the user to build, visualize, and save a kiln firing schedule
using ramp and hold segments. It is intended for stained glass fusing and similar
kiln programming workflows.

FEATURES
- Enter each segment with:
  Starting Temperature (°F)
  Target Temperature (°F)
  Rate (°F per hour)
  Hold / Soak Time (minutes)
- Automatically carries the previous Target Temperature into the next Starting Temperature
- Graphs the firing profile on a line chart
- Displays a segment table
- Shows:
  Total Segments
  Final Temperature
  Total Time
- Save Segment Table to a text file
- Help button opens this README file

HOW TO USE
1. Enter the Starting Temperature for the first segment.
2. Enter the Target Temperature.
3. Enter the Rate in °F per hour.
4. Enter the Hold / Soak Time in minutes.
5. Click Add Segment.
6. Review the graph and table.
7. Repeat for additional segments.
8. Click Save Segment Table to export the segment table.
9. Click Help to open this file.

BUTTONS
Add Segment
Adds the current segment to the program, updates the chart and table, sets the
next Starting Temperature to the current Target Temperature, and moves the cursor
back to the Target Temperature field.

Help
Opens this README.txt file in a new window or tab.

Remove Last Segment
Removes the most recently entered segment.

Save Segment Table
Exports the current segment table as a text file.

Clear All
Clears all entered segments and sets all input fields to 0.

NOTES
- All temperatures are in °F.
- Rate must be greater than 0.
- Hold time may be 0.
- Horizontal lines on the chart represent hold/soak periods.
- Sloped lines represent ramping between temperatures.
